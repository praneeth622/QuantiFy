# QuantiFy

QuantiFy is a helper analytics platform for traders and researchers at Multi-Factor Trading (MFT) firms. The platform supports analysis and research across cash and derivatives, focused on:

- Statistical arbitrage (stat-arb)
- Risk-premia harvesting
- Market-making analytics
- Term-structure analysis and calls
- Micro-alpha discovery

Covered markets include commodities, fixed income, energy, and equities.

## Features

- Signal and factor visualization
- Backtest and P&L attribution summaries
- Risk decomposition and exposure analytics
- Cross-asset term structure tooling
- Lightweight dashboard for researcher workflows

## Tech stack

- Next.js (React)
- Tailwind CSS
- TypeScript
- Recharts / Embla / Framer Motion for UI and charts

## Local development

1. Install dependencies:

```bash
pnpm install
```

2. Run the dev server:

```bash
pnpm dev
```

3. Build for production:

```bash
pnpm build
```
